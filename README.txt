Premièrement,on va créer une page HTML login.php.
Ensuite le style va ajouter la mise en forme à notre fenêtre de connexion.
Enfin,pour valider le formulaire, il faut créer un fichier pour vérifier les champs de ce dernier,mais avant il ne faut pas oubliez pas de créer la table 'utilisateur' dans la base de données.
Dernièrement,le serveur verifie si les informations fournies sont correctes,et si c'est le cas,il crée une session au nom d'utilisateur et se redirige vers la page principale.php.
Donc il faut rajouter un bouton pour se déconnecter et detruire la session en cours.
